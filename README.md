## ğŸš€ Getting Started

### Prerequisites

No special prerequisites needed! Just a web browser to view the site.

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/matteoavigni/matteoavigni.github.io.git
```

2. Navigate to the project directory:
```bash
cd matteoavigni.github.io
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

4. Visit `http://localhost:8000` in your browser
